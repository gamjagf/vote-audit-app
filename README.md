# 투표록 7개 선거 빠른 심사 앱

## 실행 방법

터미널에 붙이세요.

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 사용 방법

1. 투표록 사진을 업로드하거나 카메라로 촬영합니다.
2. 선거별 수령매수, 잔여 일련번호, 기재된 잔여매수, 교부매수, 투표자수계, 특수유형 투표자 수를 입력합니다.
3. 자동으로 다음 항목을 검산합니다.

- 잔여매수 = 끝번호 - 시작번호 + 1 + 오·훼손 미교부
- 교부매수 = 수령매수 - 잔여매수
- 투표자수계 = 교부매수
- 선거인명부등재자 = 투표자수계 - 특수유형 투표자 합계

## 주의

현재 버전은 현장 심사 보조용 MVP입니다.
OCR 자동 인식은 추후 추가 가능합니다.
